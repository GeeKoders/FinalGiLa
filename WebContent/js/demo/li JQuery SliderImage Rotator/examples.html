Examples/Skins

It never has been easier to build a website slider/banner image rotator! liJquery Image Slider/Banner comes with more than 7 different skins and option sets. Switching between the different skins is a breeze, and you only need to change a single .css file name. Hiding, showing, moving buttons and controls around is extremely simplified, and requires only changing of image names and position values.

All the skins are entirly CSS driven and the "how-to-use" learning curve is equal to zero. All the options are required upon initialization of the banner in the HTML file, so you do not need to mess up with JS coding and tweaking.

On top of all, the CSS and JS files are extremely well commented, so if you need to digg deeper into the code you'll not be left alone.
